# Data Analyst Internship – SkillCraft Technology

This repository showcases the tasks and projects completed during my **Data Analyst Internship at SkillCraft Technology**.

The internship focused on building practical skills in:
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization  
- Generating actionable business insights  

Tools and technologies used include **Excel, Python, and Power BI** to analyze datasets and support data-driven decision-making.

---

## Tasks Overview

### Task 1: Sales Dashboard (Excel)
- Created an interactive dashboard using Pivot Tables and charts
- Analyzed sales by customer segment and trends over time
- Used slicers for dynamic filtering
- Applied conditional formatting to highlight performance


### Task 2: Data Cleaning & Preprocessing
- Cleaned raw datasets by handling missing values and inconsistencies  
- Performed data transformation for better analysis  
- Prepared structured datasets for further exploration  

### Task 3: Interactive Dashboard
- Built a dynamic dashboard using HR Attrition dataset
- Created KPIs like attrition count and attrition rate
- Used filters and slicers for interactivity
- Visualized department-wise attrition insights 


### Task 4: Exploratory Data Analysis (EDA)
- Conducted detailed analysis to identify trends and patterns  
- Used visualizations to understand relationships in data  
- Generated meaningful insights for decision-making
